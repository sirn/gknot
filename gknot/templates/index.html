Gknot

Gknot is a simple web page encoding conversation service. It converts any non UTF-8 webpages to proper encoding and automatically rewrite links to its original location.

To use Gknot, simply append after this page URL in this format: 

/soup/protocol/domain/path

Or you can use this bookmarklet: Knot it!

Please note this service is run on a very limited resource. If the service is down due to quota limit, please try again the next day.

About

Gknot was written by Kridsada Thanabulpong using Mark Pilgrim's excellent chardet library for encoding detection. Source code is available under BSD license. This service runs on Google AppEngine.
